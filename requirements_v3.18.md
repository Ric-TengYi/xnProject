## 技术要求

### 数据库
- 新增表需包含：创建时间、更新时间、逻辑删除字段
- 使用 MyBatis-Plus 框架

### API 规范
- RESTful 风格
- 返回格式统一：{ code, message, data }
- 分页参数：pageNum, pageSize
- 租户隔离：所有查询需考虑 tenant_id

### 前端
- 使用 Vue3 + Element Plus
- 组件复用
- 保持现有代码风格一致

---

## 开发要求

1. **并行开发**：多个模块可同时开发，按功能模块分组
2. **先核心后辅助**：优先实现核心业务逻辑（合同、结算）
3. **可测试**：每个接口需可单独测试
4. **保持兼容**：不破坏现有功能
