## 八、结论与建议

- **通过**：Web 登录、鉴权（/api/me、orgs、roles）、用户创建（平台用户与司机端）、用户详情含 userType、合同/场地/车辆/项目列表接口均返回 200，与预期一致。
- **阻塞**：小程序登录接口在当前环境返回 401，需重启后端（或确认 JwtAuthFilter 已放行 /api/mini/auth/login）后复测。
- **建议**：  
  1) 重启后端后再次调用 POST /api/mini/auth/login，验证 driver1 返回 userType=DRIVER 及 token。  
  2) 执行 009_driver_role_seed.sql 后，角色列表可包含「司机」，便于前端「司机端」选项与角色联动。  
  3) 在 Web 与小程序启动后，按《端到端测试计划》做 UI 与司机端入口/重定向的联调测试。
