## 单模块执行步骤（建议）

对每个模块：

1. **打开用例文档**：`docs/testcases/XX-模块名-字段级测试用例.md`
2. **打开对应页面**：按上表「主要路由」在浏览器访问（已登录状态下）
3. **按查 → 增 → 改 → 删** 顺序执行表中用例，在用例文档的「实际结果」列填写
4. **回到本清单**：将该模块「执行状态」改为「已执行」，「通过/总数」填写（如 12/15），「备注」写阻塞或缺陷摘要
5. **发现问题**：在 `docs/功能测试报告.md` 或缺陷列表记录，并可在备注中写「见缺陷 #xx」
