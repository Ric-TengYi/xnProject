## 执行说明

### 前置条件
1. 后端服务运行在 `localhost:8090`
2. 数据库中存在初始租户数据（tenantId=1）
3. 管理员账号 admin 可正常登录

### 执行方式
```bash
# 先获取token
TOKEN=$(curl -s -X POST http://localhost:8090/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin"}' | python3 -c "import sys,json; print(json.load(sys.stdin)['data']['token'])")
echo $TOKEN > /tmp/xngl-test-token.txt

# 执行测试
python3 scripts/test_permission_system.py
```

### 测试数据清理
脚本会自动清理创建的测试组织。如需手动清理残留数据：
```sql
DELETE FROM sys_menu WHERE menu_code = 'TEST_FUNC_MENU';
DELETE FROM sys_permission WHERE permission_code = 'TEST_BTN_VIEW';
DELETE FROM sys_role WHERE role_code = 'TEST_FUNC_ROLE';
DELETE FROM sys_org WHERE org_code IN ('TEST_NEW_ADMIN','TEST_BIND_ADMIN','TEST_LEGACY_ADMIN');
DELETE FROM sys_user_role_rel WHERE user_id IN (SELECT id FROM sys_user WHERE username='test_custom_admin');
DELETE FROM sys_user_org_rel WHERE user_id IN (SELECT id FROM sys_user WHERE username='test_custom_admin');
DELETE FROM sys_user WHERE username = 'test_custom_admin';
```
