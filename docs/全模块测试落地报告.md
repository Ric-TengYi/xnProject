- **未执行**：L-06 错误租户；L-08～L-10 验证码登录必填/格式/接口。
- **缺陷**：无。
- **建议**：补测验证码登录与登出。

### 2. 用户/组织（04-用户组织）

- **已执行**：U-C-01～U-C-05；**U-A-01～U-A-03 新增人员**（弹窗、必填、提交 POST /api/users 成功后刷新列表）。
- **未执行**：U-U-01～U-U-02 编辑、停用；U-D-01 删除。
- **缺陷**：无。
- **建议**：补测编辑/停用。

### 3. 角色管理（05-角色管理）

- **已执行**：R-C-01 角色列表 200；R-C-02 权限树 200；R-U-02 保存权限 PUT /api/roles/:id/permissions 200（后端已接受 RolePermissionsDto）；页面加载正常（角色与权限管理、保存配置）。
- **未执行**：R-C-03 数据权限；R-A-01 新增角色；R-U-01 编辑角色；R-D-01 删除。
- **缺陷**：无。
- **建议**：补测新增/编辑/删除角色。

### 4. 数据字典（06-数据字典）

- **已执行**：D-C-01 页面加载（左侧类型、右侧表格）；D-C-02 切换类型展示；GET /api/data-dicts 当前 500（后端 data-dict 未挂载到 web 入口，前端为静态数据）。

### 10. 消纳/场地（01-场地CRUD）

- **已执行**：列表、搜索、新增、详情直链；**高级筛选**（点击打开右侧 Drawer，场地类型、状态筛选，重置/查询）。

- **建议**：卡片「更多」增加 aria-label；筛选逻辑已参与 filteredSites 计算。
