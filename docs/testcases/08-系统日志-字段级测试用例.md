# 系统日志模块 - 字段级测试用例

**页面**：/settings/logs（系统日志）

## 一、查（登录日志 + 操作日志 + 错误日志）

| 用例编号 | 测试点 | 接口/字段 | 步骤 | 预期结果 | 实际结果 |
|----------|--------|-----------|------|----------|----------|
| G-C-01 | 登录日志 Tab | GET /api/login-logs | 进入页面，Tab「登录日志」 | 表格：登录账号、登录时间、IP、终端信息、状态 | **通过** 接口 200，data.records 有 25 条，含 username、loginTime、ip、success |
| G-C-02 | 登录日志分页 | pageNo, pageSize | 切换页码 | 请求带分页参数，表格刷新 | **通过** 接口 pageNo=2&pageSize=5 返回 200，total=29，records 为第2页数据 |
| G-C-03 | 操作日志 Tab | GET /api/operation-logs | 切换至「操作日志」 | 表格：操作人、操作模块、操作类型、操作内容、操作时间、IP | **通过** 已执行 008 补丁，接口 200，data.records 可为空 |
| G-C-04 | 操作日志分页 | - | 切换页码 | 同上 | |
| G-C-05 | 错误日志 Tab | 若存在 | 切换至「错误日志」 | 有表格或占位 | **通过** 页面有 Tab：操作日志、登录日志、错误日志 |
| G-C-06 | 搜索（登录） | 搜索框 | 输入账号/IP | 请求带 keyword 或筛选参数 | 未测 |
| G-C-07 | 搜索（操作） | 搜索框 | 输入操作人/模块/内容 | 请求带 keyword | 未测 |
| G-C-08 | 时间范围 | RangePicker | 选择时间范围 | 请求带开始/结束时间 | 未测 |
