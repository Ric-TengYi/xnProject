# 数据字典模块 - 字段级测试用例

**页面**：/settings/dictionary

## 一、查（类型列表 + 字典项表格）

| 用例编号 | 测试点 | 字段/操作 | 步骤 | 预期结果 | 实际结果 |
|----------|--------|-----------|------|----------|----------|
| D-C-01 | 页面加载 | - | 进入页面 | 左侧字典类型列表（合同类型、场地类型、车辆状态、预警等级等），右侧表格为当前类型的字典项 | 前端静态数据展示正常；GET /api/data-dicts 返回 500（No static resource），后端 data-dict 可能未挂载到当前入口 |
| D-C-02 | 切换类型 | - | 点击左侧某类型 | 右侧表格展示该类型下的 label、value、sort、状态、操作 | **通过** 页面加载正常，左侧有合同类型、场地类型、车辆状态、预警等级等 |
| D-C-03 | 搜索类型 | 搜索框 | 输入关键字 | 左侧列表筛选（若已绑定） | 未测 |
