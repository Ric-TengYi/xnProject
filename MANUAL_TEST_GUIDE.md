# 多租户SaaS系统手动测试指南

## 测试环境
- 前端: http://localhost:5173
- 后端: http://localhost:8090
- 测试账号: demo / 123456

## 测试场景

### 场景1: 用户个人中心菜单
1. 登录系统
2. 点击左下角用户菜单（显示用户名）
3. **验证菜单项**：
   - ✓ 个人信息
   - ✓ 修改密码
   - ✓ 用户管理（仅TENANT_ADMIN可见）
   - ✓ 退出登录

### 场景2: 修改密码功能
1. 点击左下角用户菜单 → 修改密码
2. **验证弹窗**：
   - ✓ 弹窗标题：修改密码
   - ✓ 表单字段：原密码、新密码、确认密码
   - ✓ 验证密码一致性
3. **测试修改**：
   - 输入原密码：123456
   - 输入新密码：test123456
   - 确认密码：test123456
   - 点击确定
   - ✓ 应显示成功提示

### 场景3: 角色管理分页
1. 进入角色管理页面
2. **验证左侧角色列表**：
   - ✓ 显示分页控件
   - ✓ 默认显示20条记录
   - ✓ 能翻页加载更多
   - ✓ 列表不超出页面边界

### 场景4: 用户管理隔离
1. 点击左下角用户菜单 → 用户管理
2. **验证用户列表**：
   - ✓ 只显示当前租户的用户
   - ✓ 支持分页（每页20条）
   - ✓ 列表显示正确

### 场景5: 页面标签导航
1. 进入多个页面（如角色管理、用户管理）
2. **验证页面标签**：
   - ✓ 顶部显示页面标签
   - ✓ 能点击标签切换页面
   - ✓ 能关闭标签（X按钮）

### 场景6: 个人信息查看
1. 点击左下角用户菜单 → 个人信息
2. **验证弹窗显示**：
   - ✓ 用户名：demo
   - ✓ 姓名：Demo Admin
   - ✓ 用户类型：TENANT_ADMIN
